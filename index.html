What's a NEOTOKYO?

NEOTOKYO is a Sourcemod developed by Studio Radi-8 back in 2009, with a Steam release in 2014. The game is widely considered to be dead, but the more accurate description is 'on community life support'. BonAHNSa is basically the game's blood bag. NEOTOKYO's community is small, yet dedicated in keeping the game alive. BonAHNSa currently hosts the only other updated server for NEOTOKYO.

New to NEOTOKYO?

If you just happened to stumble on this site not knowing about BonAHNSa or NEOTOKYO and got interested in playing, welcome to NEOTOKYO!

To start off, you need to install the actual game from Steam. It's free! After its done downloading, you should follow our Guide to making the game crash resistant and follow up on the game mechanics. After that, feel free to join the Discord!

We play every Friday between 1800-2200 UTC, and more commonly along the weekend. Have fun!
